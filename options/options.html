How your data is handled

By default, your question is sent (with your access code) to a small proxy server which holds the shared Groq key and forwards the request to Groq. The key itself never reaches your browser or this extension's code, and your access code only identifies you to that proxy, not to Groq. If you enter your own API key, your browser talks directly to that provider instead, bypassing the proxy entirely, and the key is stored only in this browser (chrome.storage.local). Page content is sent only when you ask a question. Nothing is sent to NaviMind's authors.
